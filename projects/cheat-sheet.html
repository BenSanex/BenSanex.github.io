My Selection of Simple Ruby Methods

These aren't in any particular order, they're just methods I've used multiple times throughout Phase 0 and are are good for quick references.

If..else

Checks to see if condition is true

If condition
elsif optional condition
else
end
While

Executes a code while a condition is true

while condition do
code
end
for...

Loops through code once for each element

for i in [array or range]
i something
end
each

executes code once for each element, especially in arrays

a.each
{ |item| print item }
case

similiar to if statement with many conditions

case [variable]
when (variable ==) something
code
when (variable ==) something else
more code
else
if none are true code
end
#length, #count

Determines number of items in or length of an object

'string'.length
=> 6
#reverse(!)

reverses an array or string

'string'.reverse!
=>'gnirts'
#sort(!)

sorts an array

['b','a','c'].sort!
=> ['a','b','c']
#join

Creates a string from an array separated by an arguement

['a','b','c'].join(' ')
=> 'a b c'
#map(!), #collect(!)

Executes code on each object of an array

['a','b','c'].collect!
{ |item| item + '!'}
=> ['a!','b!','c!']
#flatten

Takes multi level array down argument levels

[['a','b'],['c','d']].flatten()
=> ['a','b','c','d']
